# Unit Limit

## Summary

Unit Limit is the army-construction limit on how many copies of each unit can be included in an army.

## Definition

Unit Limit is the maximum number of each unit that can be included in an army at the selected battle size.

## Rules

Published examples:

| Battle Size | Unit Limit |
| --- | ---: |
| Incursion | 2 |
| Strike Force | 3 |

The published note states that the Unit Limit is doubled for BATTLELINE units.

## Interactions

- [[Battle Size]] determines the unit limit.
- [[Keywords]] determine whether a unit has the BATTLELINE keyword.
- [[Army Construction]] applies the unit limit before the pre-battle sequence begins.

# Detachment Points

## Summary

Detachment Points are the army-construction budget used to select detachments for an army list.

They are the primary limit on how detachments can be combined.

## Definition

Detachment Points are the list-building points available for choosing detachments.

They are not [[Command Point (CP)]]. Command Points are spent during the game to use stratagems; Detachment Points are used before the game while constructing the army.

## Rules

Depending on battle size, a player has a number of Detachment Points available.

Each detachment has a Detachment Point cost. The source describes this cost as ranging from 1 for gentler or narrower detachment rules to 3 for detachments that affect the whole army with significant boosts.

Published examples:

| Battle Size | Detachment Points |
| --- | ---: |
| Incursion | 2 |
| Strike Force | 3 |

## Combining Detachments

Detachments can be combined if their total Detachment Point cost does not exceed the army's Detachment Point budget.

For a Strike Force army with 3 Detachment Points, the common legal patterns are:

| Pattern | Legal? | Reason |
| --- | --- | --- |
| 1DP + 1DP + 1DP | Yes | Total cost is 3DP. |
| 2DP + 1DP | Yes | Total cost is 3DP. |
| 3DP | Yes | Total cost is 3DP. |
| 2DP + 2DP | No | Total cost is 4DP. |
| 3DP + 1DP | No | Total cost is 4DP. |

A detachment's [[Force Dispositions|Force Disposition]] category is not an additional cost and is not a restriction on combining detachments. It matters later for mission selection.

## Interactions

- [[Battle Size]] determines how many Detachment Points are available.
- [[Detachments]] consume Detachment Points.
- [[Army Construction]] uses Detachment Points before the battle begins.
- [[Force Dispositions]] are unlocked by detachments but do not spend Detachment Points.
- [[Stratagem System]] is related only because detachments may grant stratagems.

## Restrictions

- Detachment Points are not generated during the Command phase.
- Detachment Points are not spent to use stratagems during the game.
- Detachment Points limit the total cost of selected detachments, not the number of Force Disposition categories represented.
- This article does not define faction-specific detachment costs unless a source-backed detachment article is created.

# Using Stratagems

## Summary

Using Stratagems is the umbrella rule for spending CP to use stratagems and resolving their target selection, payment, and effects.

## Definition

Players spend CP to use stratagems, resolving target selection, CP payment, then the stratagem's effects.

## Rules

Every stratagem states its CP cost, WHEN, TARGET, EFFECT, and any RESTRICTIONS.

A player cannot use the same stratagem more than once in the same phase.

Unless otherwise stated, a player cannot target the same unit with more than one stratagem in the same phase.

Additional optional sections with extra CP cost can be used only if the player increases the total CP cost accordingly.

If the player does not have enough CP, the stratagem cannot be used.

## Interactions

- [[Command Point (CP)]] is the glossary resource that this rule spends.
- [[Battle-shock Rolls]] matters because battle-shocked units can constrain stratagem targeting elsewhere in the system.
- [[Epic Challenge]] and [[Fire Overwatch]] are published child stratagem procedures under this umbrella.
- [[Stratagem System]] is the parent system map for the domain.

## Restrictions

- The same stratagem cannot be used more than once in the same phase.
- Unless otherwise stated, the same unit cannot be targeted by more than one stratagem in the same phase.
- If the player does not have enough CP, the stratagem cannot be used.

# Command Point (CP)

## Summary

Command Point is the resource spent to use stratagems.

## Definition

Resource spent to use stratagems.

## Rules

Command Points are used in the stratagem framework.

Gain Core CP gives each player 1 CP per turn; they are Command Points like any other.

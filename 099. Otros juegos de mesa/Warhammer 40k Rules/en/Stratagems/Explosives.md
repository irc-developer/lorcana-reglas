# Explosives

## Summary

Explosives is the core stratagem that lets an eligible EXPLOSIVES or GRENADES unit inflict mortal wounds on a nearby visible enemy unit.

## Definition

Explosives is used in your Shooting phase on an eligible friendly EXPLOSIVES or GRENADES unit.

## Rules

Cost: 1CP.

The target unit must be friendly, unengaged, eligible to shoot, and must not have made an advance move this turn.

Select one EXPLOSIVES or GRENADES model in that unit.

Select one visible unengaged enemy unit within 8 inches of that model.

Roll six D6; each 4+ causes that enemy unit to suffer 1 mortal wound.

## Interactions

- [[Shooting Phase]] provides the timing window.
- [[Mortal Wounds]] owns the damage type inflicted by the stratagem.
- [[Using Stratagems]] governs the CP and target framework.

## Restrictions

- Explosives does not make the selected unit shoot using Making Attacks.
- The enemy unit must be both visible and within 8 inches of the selected model.

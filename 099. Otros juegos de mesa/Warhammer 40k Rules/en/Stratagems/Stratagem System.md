# Stratagem System

## Overview

Stratagem System is the primary entry point for the Stratagems domain. It maps the CP-spending umbrella rule and the cross-domain stratagem procedures that modify movement, shooting, charge, fight, terrain, and reserves interactions.

## Scope

This domain covers the stratagem framework, the planned command-point glossary support, and the planned stratagem procedures that modify multiple other systems without replacing their owned mechanics.

It helps answer questions about where stratagem rules belong, which parts of the CP-spending framework this domain owns, and when a lookup should move into battle-shock, movement, shooting, charge, fight, terrain, or reserves systems for the underlying mechanic.

## Domain Ownership

- Owner domain: Stratagems.
- Publication role: organise the core stratagem framework and the individual stratagem procedures that alter multiple other systems.
- Domain system role: provide the authoritative entry point and navigation surface for planned Stratagem-domain content.

## Domain Boundaries

Stratagems owns:

- the planned framework article Using Stratagems.
- the planned stratagem procedures Command Re-roll, Epic Challenge, Insane Bravery, Explosives, Crushing Impact, Rapid Ingress, Fire Overwatch, Smokescreen, Heroic Intervention, and Counteroffensive.
- the planned glossary support surface Command Point (CP).

Stratagems does not own:

- [[Battle-shock System]] ownership of battle-shock state rules, even where that state constrains stratagem targeting.
- [[Movement System]], [[Shooting System]], [[Charge System]], and [[Fight System]] ownership of the underlying phase mechanics that stratagems modify.
- [[Terrain Visibility System]] ownership of terrain and visibility rules affected by some stratagems.
- [[Strategic Reserves System]] ownership of reserve handling even where Rapid Ingress interacts with that domain.
- [[Battle Sequence System]] ownership of the turn and phase timing spine that stratagem timing relies on.

## Related Systems

- [[Battle Sequence System]] provides the timing windows in which stratagems are used.
- [[Battle-shock System]] constrains whether some units can be targeted by their controlling player's stratagems.
- [[Movement System]], [[Shooting System]], [[Charge System]], and [[Fight System]] are the main phase-domain systems that stratagem procedures modify.
- [[Terrain Visibility System]] provides an adjacent targeting and protection context used by some stratagems.
- [[Strategic Reserves System]] is a direct neighbouring system because Rapid Ingress modifies reserve arrival.

## Navigation

- Start here when the question is about spending CP or applying one of the core stratagem procedures.
- Read [[Using Stratagems]] when the question is about the shared CP-spending framework.
- Read [[Command Point (CP)]] when the lookup is about the resource itself.
- Read [[Epic Challenge]] or [[Fire Overwatch]] when the question has narrowed to an attack-facing published stratagem.
- Move to [[Battle Sequence System]] when the real question is about timing windows rather than stratagem ownership.
- Move to [[Battle-shock System]] when the issue is whether a unit can be targeted by a stratagem at all.
- Move to [[Strategic Reserves System]] when the lookup is specifically about Rapid Ingress.
- Move to [[Shooting System]], [[Charge System]], or [[Fight System]] when the question is really about the underlying phase mechanic modified by a stratagem.

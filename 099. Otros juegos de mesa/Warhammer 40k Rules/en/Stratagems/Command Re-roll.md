# Command Re-roll

## Summary

Command Re-roll is the core stratagem that lets a player spend 1CP to re-roll one specified roll for a friendly unit or model.

## Definition

Command Re-roll is used just after making one of the listed rolls for a friendly unit or model.

## Rules

Cost: 1CP.

Timing: any phase, just after making one of the listed rolls.

Eligible rolls include advance, charge, damage, hazard, hit, save, wound, and a roll to determine the number of attacks generated by a weapon.

Re-roll that roll.

If more than one dice is being rolled together, select one of those dice to re-roll, except charge rolls, which must be re-rolled in full.

## Interactions

- [[Using Stratagems]] governs CP spending and targeting.
- [[Dice]] provides the shared roll language.
- [[Charge Move]] is relevant because charge rolls are re-rolled in full.

## Restrictions

- This stratagem can re-roll only the listed roll types.
- For multi-dice rolls other than charge rolls, only one selected dice is re-rolled.

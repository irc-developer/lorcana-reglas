# Insane Bravery

## Summary

Insane Bravery is the once-per-battle core stratagem that makes one friendly unit automatically pass a battle-shock roll.

## Definition

Insane Bravery is used in the Battle-shock step of your Command phase just before making a battle-shock roll for a friendly unit.

## Rules

Cost: 1CP.

Target: the friendly unit about to make the battle-shock roll.

Effect: that battle-shock roll is automatically successful.

Restriction: this stratagem cannot be used more than once per battle.

## Interactions

- [[Battle-shock Rolls]] defines the roll being replaced by automatic success.
- [[Command Phase Battle-shock]] provides the main timing window.
- [[Using Stratagems]] provides the CP and usage framework.

## Restrictions

- This stratagem applies to one battle-shock roll.
- It is limited to once per battle.

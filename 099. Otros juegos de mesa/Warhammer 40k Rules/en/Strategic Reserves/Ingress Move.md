# Ingress Move

## Summary

Ingress Move is the set-up move that places a unit from [[Strategic Reserves]] onto the battlefield.

## Definition

An ingress move sets up a unit from [[Strategic Reserves]] wholly within 6" of one or more battlefield edges and more than 8" horizontally from all enemy units.

## Rules

An ingress move is a set-up move, not a normal battlefield move sequence.

Before the third [[Battle Round]], a unit making an ingress move cannot be set up within the opponent's [[Deployment zone]] unless another rule allows it.

After making an ingress move, that unit cannot make another type of move until the start of the next [[Charge Phase]] unless another rule states otherwise.

Units embarked within TRANSPORTS that are themselves in [[Strategic Reserves]] are not eligible to make [[Ingress Move|ingress moves]] directly.

## Interactions

- [[Battle Round]] provides the round-number timing that limits early reserve arrival.
- [[Charge Phase]] is the next named phase that unlocks later movement after an ingress move.
- [[Strategic Reserves]] supplies the reserve state from which an [[Ingress Move]] is made.
- Deep Strike changes where an otherwise valid [[Ingress Move]] can be set up.
- Rapid Ingress changes when an eligible unit can make an [[Ingress Move]].
- [[Repositioned Units]] return to the battlefield through the same arrival move unless another rule says otherwise.

## Restrictions

- The unit must be set up wholly within 6" of one or more battlefield edges.
- The unit must be more than 8" horizontally from all enemy units.
- Before the third [[Battle Round]], the unit cannot be set up within the opponent's [[Deployment zone]] unless another rule allows it.
- After making an ingress move, the unit cannot make another type of move until the start of the next [[Charge Phase]] unless another rule states otherwise.
- Embarked units inside transports that are in [[Strategic Reserves]] cannot make [[Ingress Move|ingress moves]] directly.

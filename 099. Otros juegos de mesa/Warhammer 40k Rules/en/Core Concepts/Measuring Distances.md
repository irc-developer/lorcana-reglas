# Measuring Distances

## Summary

Measuring Distances is the foundational Core Concepts article for inch-based distance measurement used by setup, movement, objective range, and terrain-driven thresholds.

## Definition

Distances are measured in inches and may be measured at any time.

## Rules

Unless another rule states otherwise, measure to or from the closest part of a model's base.

If a rule uses FRAME measurement, use the closest point on the model instead of the default base-measuring rule.

This article establishes how distance is measured. It does not add a separate timing restriction for when players may check distance.

## Interactions

- [[Set Up]] uses this measurement baseline when placing units on the battlefield.
- [[Movement System]] consumes this article when move distances, vertical movement, and legal end positions depend on measured distance.
- [[Objective Control System]] consumes this article when rules check whether models are within range of an objective.
- [[Terrain Visibility System]] consumes this article when terrain-owned thresholds such as detection range depend on measured distance.
- [[Move Characteristic]] provides one of the main printed values that later movement rules compare against measured distance.

## Restrictions

- This article does not define move procedures, charge distance resolution, coherency, or engagement.
- This article does not redefine special measurement exceptions beyond the rule that explicitly says to use FRAME measurement.
- If a more specific rule changes what point is measured, that specific rule takes precedence.

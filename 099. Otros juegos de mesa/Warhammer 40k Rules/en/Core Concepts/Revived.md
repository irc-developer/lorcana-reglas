# Revived

## Summary

Revived is the core-concepts rule that defines how destroyed models are returned to their unit and how that return reconnects the unit to its existing state without changing its original starting-strength reference.

## Definition

When a rule revives, resurrects, or returns models, destroyed models are added back to the unit up to its starting strength.

A revived model is a destroyed model that has been returned to its unit by such a rule.

Core Rules do not define a separate revived-unit state; the unit remains the same unit while one or more destroyed models are returned to it.

## Rules

### Returned Models

Destroyed models are returned only to their own unit.

Returned models are added back only up to that unit's starting strength.

Returned models have all wargear and enhancements they started the battle with unless a rule states otherwise.

Returned models are added with full wounds unless a rule states otherwise.

### Placement And Unit State

Returned models must be set up in coherency with models from that unit that started that phase on the battlefield.

Returned models can be engaged only if the unit is already engaged with those enemy units.

If a revived leader or support model is part of an [[Attached unit]], it remains part of that attached unit and must return to it if possible.

### Relationship To Destroyed And Strength States

Revived does not define when a model or unit is destroyed; that timing and state belong to [[Destroyed]].

Revived returns models that were already destroyed, and it does not restate the separate procedures that caused their destruction.

When models are returned, the unit's current remaining models, or a single-model unit's current remaining wounds, are re-evaluated against the same fixed reference defined by [[Starting Strength and Half-strength]].

Revived cannot take a unit above that starting-strength reference.

## Interactions

- [[Destroyed]] defines the destroyed state that must already apply before a model can be returned by a revive rule.
- [[Starting Strength and Half-strength]] defines the fixed reference that still governs below starting strength, at half-strength, and below half-strength after models are returned.
- [[Attached unit]] is relevant only when a returned leader or support model must reconnect to its existing attached unit.

## Restrictions

- This article defines only the return of destroyed models to their unit and the immediate consequences of that return.
- This article does not restate [[Destroyed]], Inflict Damage, abilities that cause models to return, or the general attached-unit rules.
- Returned models cannot be added above the unit's starting strength.
- Returned models return with the wargear, enhancements, and wound state stated by the core rule unless another rule states otherwise.

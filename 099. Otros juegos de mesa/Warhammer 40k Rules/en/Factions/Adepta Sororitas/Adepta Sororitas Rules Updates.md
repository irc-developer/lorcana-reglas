# Adepta Sororitas Rules Updates

## Summary

Adepta Sororitas Rules Updates records the non-detachment surfaces in the official version 1.1 faction pack: included datasheets, updates, errata, and FAQs.

Use this page as integration navigation, not as a replacement for the official PDF, Codex, app, or complete datasheet cards.

## Included Datasheets

The version 1.1 faction pack includes these non-Legends datasheets:

| Datasheet | Main hooks |
| --- | --- |
| Sanctifiers | INFANTRY, GRENADES, Acts of Faith, Scouts 6", Ministorum Priest attachment substitution, Cherub, Simulacrum Imperialis, and model return through Salvationist Medikit. |
| Celestian Insidiants | INFANTRY, GRENADES, Acts of Faith, defence against Psychic Attacks and [[Mortal Wounds]], quarry selection, [[Precision]], Hit roll re-rolls against the quarry, and Character attachment substitution for Dominion Squad leaders. |

The PDF also includes Legends datasheets from page 9 onward. Following the Grey Knights pattern, this workflow did not create individual datasheet articles.

## Updates And Errata

| Area | Update |
| --- | --- |
| Divine Intervention Stratagem, Effect section | Changes the Miracle dice discard and model return process. |
| Suffering and Sacrifice Stratagem, Effect section | Changes enemy target-selection obligation for enemy models within Engagement Range. |
| Devout Fanaticism Stratagem, When section | Changes the timing to the opponent's Shooting phase, just after an enemy unit has shot. |
| Immolator Transport section | Changes the Battle Sisters Squad, Dominion Squad, or Sisters Novitiate Squad splitting process at the start of the Declare Battle Formations step. |

## FAQ

| Resolved question | Active answer |
| --- | --- |
| Imagifier and Miracle dice gained at the end of a phase while using the Balance Dataslate | Yes, Litany of Deeds can re-roll the result if its conditions are met. |
| Saintly Example and Imagifier | Yes, Litany of Deeds can re-roll the additional D3 Miracle dice results before they are added to the pool. |
| Verse of Holy Piety and Vow of Atonement if the CHARACTER dies before the battle round ends | Yes, the Vow remains active for the remaining models until the start of the next battle round. |
| Using a Miracle dice as part of a re-roll | Yes. |
| Re-rolling a roll where a dice was substituted with a Miracle dice | No, that Miracle dice is lost; the unit still counts as having performed an Act of Faith. |
| Triumph of Saint Katherine and Simulacrum of the Ebon Chalice | It cannot substitute two dice in a single roll. |
| Timing for gaining a Miracle dice from Acts of Faith when an Adepta Sororitas unit is destroyed | The dice is gained when the unit is destroyed. |

# Grey Knights Rules Updates

## Summary

Grey Knights Rules Updates records the faction-pack version 1.0 updates that modify existing Grey Knights detachments, enhancements, abilities, and datasheet keywords.

Use this article as an errata navigation note, not as a replacement for the official faction pack or app.

## Detachment Updates

| Area | Update |
| --- | --- |
| Brotherhood Strike Detachment | Combat Manifestation changes its Effect distance from 3" to 6". |
| Hallowed Conclave Detachment | Eye of the Augurium lets the relevant Grey Knights unit use [[Heroic Intervention]] despite other uses of that stratagem in the phase; that use is discounted and does not block further uses on other units. |

## Datasheet Updates

| Unit or group | Update |
| --- | --- |
| Stormtalon Gunship and Stormhawk Interceptor | Remove [[Hover]] from Core Abilities; change Move and Objective Control to `-`. |
| Stormraven Gunship | Change Move to 14" and remove the [[Aircraft]] keyword. |
| Brotherhood Chaplain | Add [[Leader]] to Core Abilities. |
| Brotherhood Techmarine | Guardians of the Machine gains a Heroic Intervention hook when an enemy ends a charge move engaged with a friendly Grey Knights Vehicle within range of the Techmarine. |
| Land Raider, Land Raider Crusader, Land Raider Redeemer, Razorback, Rhino, Stormhawk Interceptor, Stormraven Gunship, Stormtalon Gunship | Add the Frame keyword to the Keywords section. |

## Integration Notes

- [[Heroic Intervention]] is the most important core stratagem cross-reference.
- [[Aircraft]] and [[Hover]] matter because several vehicle keyword profiles are changed; Frame is added as a vehicle keyword surface.
- [[Leader]] matters because it changes how the Brotherhood Chaplain participates in attached-unit construction.

# Emergency Disembark Move

## Summary

Emergency Disembark Move is the transport exit move required for an embarked unit when its TRANSPORT has just been destroyed.

## Definition

An Emergency Disembark Move sets up a unit from a TRANSPORT model that was just destroyed.

## Rules

The set-up distance is 6 inches.

Before moving, make a hazard roll for each model in the unit.

Set up each model wholly within the set-up distance of the destroyed TRANSPORT and as close as possible to it.

Each model that cannot be set up this way is destroyed.

After the move, the unit is battle-shocked and cannot declare a charge until the end of the turn.

## Interactions

- [[Disembarking]] defines when this emergency branch is required.
- [[Set Up]] provides the general set-up framework.
- [[Hazard Rolls]] resolve the risk to the passengers.
- [[Battle-shock Rolls]] and [[Battle-shocked]] explain the resulting state.

## Restrictions

- This procedure applies only when the unit was embarked within a TRANSPORT model that was just destroyed.
- The TRANSPORT is not removed from the battlefield until the emergency disembark requirement has been resolved.

# Transport Capacity

## Summary

Transport Capacity is the datasheet-listed limit that determines which friendly models can embark within a TRANSPORT and how many it can carry.

## Definition

A TRANSPORT model has a transport capacity that states the type and maximum number of friendly models eligible to embark within it.

## Rules

Transport capacity is checked against the whole unit that wants to embark.

A TRANSPORT can contain more than one unit at the same time if enough capacity remains.

Before the battle, units can start embarked only if the TRANSPORT has sufficient capacity remaining for the whole unit.

## Interactions

- [[Transports]] is the umbrella transport article.
- [[Embarking]] checks remaining transport capacity after the first battle round has started.
- [[Embarked]] is the state entered once the unit is inside the TRANSPORT.

## Restrictions

- Transport capacity is not a general points limit and does not itself move or set up units.

# Embarking

## Summary

Embarking is the procedure that lets a friendly unit enter a friendly TRANSPORT after making a normal, advance, or fall-back move.

## Definition

After the first battle round has started, an eligible friendly unit can embark within a friendly TRANSPORT model after making a normal, advance, or fall-back move.

## Rules

Every model in the embarking unit must be within 3 inches of the TRANSPORT.

The unit must not have been set up on the battlefield that turn.

The unit must be eligible to embark within that TRANSPORT as described on the TRANSPORT datasheet.

The TRANSPORT must have enough remaining transport capacity for every model in the unit.

When the unit embarks, the active player removes it from the battlefield and places it aside; the unit is now embarked and is not on the battlefield.

## Interactions

- [[Transport Capacity]] provides the capacity check.
- [[Embarked]] describes the state entered after the unit is removed from the battlefield.
- [[Normal Move]], [[Advance Move]], and [[Fall-back Move]] are the movement procedures that can lead into embarking.

## Restrictions

- A unit that was set up on the battlefield this turn cannot embark using this rule.
- Embarking removes the unit from the battlefield; it does not set the unit up elsewhere.

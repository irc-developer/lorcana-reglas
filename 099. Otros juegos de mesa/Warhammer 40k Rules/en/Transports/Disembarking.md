# Disembarking

## Summary

Disembarking is the umbrella rule for units leaving a TRANSPORT either during the active player's Movement phase or when the TRANSPORT is destroyed.

## Definition

A friendly unit embarked within a TRANSPORT can leave it by making a disembark move when the rules allow it.

## Rules

In the active player's Movement phase, each friendly unit embarked within a TRANSPORT model can disembark by making a [[Disembark Move]].

If a TRANSPORT model is destroyed, before it is removed from the battlefield, the active player must make an [[Emergency Disembark Move]] with each unit embarked within it.

## Interactions

- [[Disembark Move]] owns the normal Movement phase exit branches.
- [[Emergency Disembark Move]] owns the destroyed-transport exit branch.
- [[Embarked]] is the state being exited.
- [[Movement Phase]] provides the normal phase window.

## Restrictions

- This article does not replace the specific rules of the two disembark move types.

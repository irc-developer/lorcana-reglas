# Surge Move

## Summary

Surge Move is the triggered movement procedure that moves an eligible unengaged, non-battle-shocked unit toward the closest enemy unit.

## Definition

A Surge Move is a move type with a maximum distance defined by the rule that allowed the move.

## Rules

The rule allowing the Surge Move must have been triggered.

The unit must not be battle-shocked, must be unengaged, and must not have moved this phase.

Before moving, select the closest enemy unit as the surge target.

Each model must end engaged with the surge target if possible.

A model that cannot end engaged with the surge target must end as close as possible to it.

After moving, the unit cannot be engaged with enemy units that were not the surge target and cannot move again this phase.

## Interactions

- [[Moving Units]] provides the general movement framework.
- [[Battle-shock Rolls]] and [[Battle-shocked]] matter because battle-shocked units are not eligible.
- [[Engagement]] defines the engaged and unengaged states used by the move.

## Restrictions

- A Surge Move requires another rule to grant it.
- The unit cannot use a Surge Move if it has already moved this phase.

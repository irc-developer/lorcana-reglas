# Light

## Summary

Light is the terrain category for features that can provide cover while remaining traversable and while contributing to obscuring terrain areas.

## Definition

Light terrain can provide cover from incoming attacks but does not significantly impede movement or provide lasting defence.

## Rules

Examples given in the rules include barricades, low walls, and statuary.

All models can move horizontally and vertically through light terrain features.

Terrain areas containing one or more light terrain features are [[Obscuring terrain area|obscuring terrain areas]].

## Interactions

- [[Terrain Categories]] classifies Light as one of the terrain categories.
- [[Benefit of Cover]] uses Light terrain as one of the common terrain states that can protect a target.
- [[Obscuring]] uses Light terrain when determining whether a terrain area blocks visibility between models.
- [[Movement System]] consumes the terrain effect, but Movement still owns the procedure of moving.

## Restrictions

- Light does not itself resolve attacks or determine hit, wound, or save procedure.
- Light does not define the movement procedure; it only states the terrain effect that movement procedures consume.
- Light should not be treated as a synonym for all cover-producing terrain because the exact cover test belongs to [[Benefit of Cover]].

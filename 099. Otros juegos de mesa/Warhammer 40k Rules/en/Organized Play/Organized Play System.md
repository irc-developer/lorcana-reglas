# Organized Play System

> This article describes tournament or organized play procedures. It does not replace the Warhammer 40,000 Core Rules.

## Summary

The Organized Play System is the navigation hub for official event procedures drawn from the Warhammer Event Companion and its Doubles, Teams, and Dominatus companions.

## Scope

This domain covers event preparation, mission procedure, scoring, pairing, ranking, rules resources, and specialist event formats. It excludes layouts, maps, terrain diagrams, base sizes, and any policy not stated in the source packs.

## Practical Use

Start with [[Tournament Rules]], then read the preparation, mission, scoring, and pairing articles that apply to the event. If the event is Doubles, Teams, or Dominatus, also apply the relevant specialist article.

## Player Impact

Players should identify the event format, read its event pack, prepare the required roster information, and follow the event-specific mission and scoring procedure.

## Organizer Impact

Organizers should publish any adaptations before play, identify which companion applies, and avoid presenting event procedures as changes to the Core Rules outside that event.

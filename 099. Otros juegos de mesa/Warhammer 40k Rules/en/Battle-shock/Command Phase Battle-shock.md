# Command Phase Battle-shock

## Summary

Command Phase Battle-shock is the command-phase application of the Battle-shock system that determines which units must make battle-shock rolls at that timing and how already battle-shocked units can recover.

## Definition

In the active player's Command phase, battle-shock rolls are made for units that are currently battle-shocked and/or at or below half-strength.

## Rules

During the active player's Command phase, units that are currently battle-shocked and units that are at or below half-strength make battle-shock rolls.

If a currently battle-shocked unit succeeds on that roll, it becomes no longer battle-shocked.

This timing applies the battle-shock procedure; it does not replace the underlying rule that defines success, failure, or the state created on failure.

## Interactions

- [[Battle-shock Rolls]] defines the roll being made and the consequences of success or failure.
- [[Battle-shocked]] is the state that can persist into this step and can be removed here on a successful roll.
- [[Starting Strength and Half-strength]] provides the half-strength condition that makes many units eligible to roll in this step.
- [[Command Phase]] is the canonical Game Sequence timing article that owns the phase containing this application.
- [[Battle Sequence System]] provides the wider timing spine above that phase-level owner.
- [[Stratagem System]] remains the owner of Insane Bravery as the stratagem-side override that can make one of these rolls automatically successful.

## Restrictions

- This article does not redefine the Command phase or broader turn structure.
- This article does not redefine the battle-shock roll itself; it applies that rule at a specific timing window.
- This article does not absorb Insane Bravery or general stratagem rules into Battle-shock.

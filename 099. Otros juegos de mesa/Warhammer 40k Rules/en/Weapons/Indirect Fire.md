# Indirect Fire

## Summary

Indirect Fire is the weapon ability that allows a unit with one or more qualifying weapons to use indirect shooting.

## Definition

Units with one or more Indirect Fire weapons can use indirect shooting.

## Rules

Indirect Fire grants access to [[Indirect Shooting]].

It does not itself redefine the shooting-mode procedure that follows.

## Interactions

- [[Indirect Shooting]] is the shooting mode enabled by this weapon ability.
- [[Shooting System]] owns the phase-level procedure that consumes this ability.
- [[Weapon Abilities]] places Indirect Fire inside the broader weapon-ability cluster.
- [[Core Abilities]] provides the wider attack-facing ability index.
- [[Attack System]] is the shared downstream pipeline used once the mode has been chosen.

## Restrictions

- Indirect Fire does not by itself resolve target visibility, hit-roll penalties, or cover effects.
- This page is distinct from [[Indirect Shooting]], which is the enabled shooting mode rather than the weapon ability.

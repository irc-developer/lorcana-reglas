# Precision

## Summary

Precision is the weapon ability that lets the active player redirect the current allocation group to a visible CHARACTER allocation group at the start of Allocation Order.

## Definition

At the start of Allocation Order, the active player can make a visible CHARACTER allocation group the current allocation group while resolving those attacks.

## Rules

Precision applies at the start of Allocation Order during [[Save Rolls]].

While resolving those attacks, the active player can make a visible CHARACTER allocation group the current allocation group.

## Interactions

- [[Save Rolls]] is the attack step that contains the allocation-order timing point this ability modifies.
- [[Attacking Attached Units]] is the neighbouring attached-unit rule that interacts with this allocation override.
- [[Attack System]] is the shared attack pipeline that this ability modifies.
- [[Weapon Abilities]] places Precision inside the broader weapon-ability cluster.

## Restrictions

- Precision applies only at the start of Allocation Order.
- The target allocation group must be a visible CHARACTER allocation group.
- This page does not redefine the wider save or attached-unit procedures.

# Rapid Fire

## Summary

Rapid Fire is the weapon ability that adds attack dice if the target was within half range in the Select Targets step.

## Definition

Add X attack dice if the target was within half range in the Select Targets step.

## Rules

Rapid Fire checks whether the target was within half range during [[Select Targets]].

When that condition is met, add the listed extra attack dice in [[Resolve Attacks]].

## Interactions

- [[Select Targets]] is the upstream step that fixes whether half range was met.
- [[Resolve Attacks]] is the attack-declaration step that receives the extra attack dice.
- [[Weapon Abilities]] places Rapid Fire inside the broader weapon-ability cluster.
- [[Core Abilities]] provides the wider attack-facing ability index.

## Restrictions

- Rapid Fire applies only if the target was within half range in Select Targets.
- Rapid Fire changes attack-dice gathering only.

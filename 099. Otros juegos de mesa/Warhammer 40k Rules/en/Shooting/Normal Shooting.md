# Normal Shooting

## Summary

Normal Shooting is the default ranged-combat mode for an unengaged unit that did not advance.

## Definition

An unengaged unit that did not advance shoots using the normal attack rules.

## Rules

Normal Shooting is one of the eligible shooting types used when a unit is selected to [[Shoot]].

When a unit uses Normal Shooting, it resolves its attacks through [[Making Attacks]].

After shooting, the unit cannot start an action until the end of the phase.

## Interactions

- [[Shoot]] is the parent procedure that selects the unit and applies one eligible shooting type.
- [[Making Attacks]] and [[Attack System]] provide the shared attack pipeline used once this shooting mode is allowed.
- This mode is the baseline shooting branch that sits alongside the other published shooting exceptions in the domain.

## Restrictions

- The unit must be unengaged.
- The unit must not have advanced this turn.
- This article does not redefine target selection, weapon selection, hit rolls, or any other shared attack-resolution step.

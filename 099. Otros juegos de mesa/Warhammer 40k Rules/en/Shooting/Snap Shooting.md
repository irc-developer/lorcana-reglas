# Snap Shooting

## Summary

Snap Shooting is the special shooting procedure used only when another rule gives permission to shoot in that way, most notably outside the normal Shooting phase flow.

## Definition

Snap Shooting is a special shooting type used when another rule permits it.

## Rules

Snap Shooting can only target one visible enemy unit within 24 inches, and only if that unit is otherwise an eligible target.

Each attack only hits on an unmodified 6, regardless of Ballistic Skill or modifiers.

Hit rolls cannot be re-rolled.

After shooting, the unit cannot start an action until the end of the phase.

## Interactions

- [[Shooting System]] maps this procedure as one of the Shooting domain's published surfaces.
- [[Attack System]] and [[Hit Rolls]] provide the shared attack and hit-roll framework that this procedure sharply modifies.
- [[Visible]] remains the published glossary target for the visibility requirement consumed by this procedure.
- [[Stratagem System]] is the published neighbouring system for external permissions that can instruct a unit to use this shooting type.

## Restrictions

- Snap Shooting only exists when another rule grants permission to use it.
- It can target only one enemy unit.
- The target must be visible and within 24 inches.
- Hit rolls made through this procedure cannot be re-rolled.
- This procedure does not define the external permission that authorises it.

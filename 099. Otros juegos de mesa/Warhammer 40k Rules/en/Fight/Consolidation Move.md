# Consolidation Move

## Summary

Consolidation Move is the fight-owned positional procedure that lets an eligible unit move up to 3 inches after fighting by following the mandatory consolidation mode that currently applies.

## Definition

After fighting, eligible units can move up to 3 inches using one mandatory consolidation mode.

## Rules

If the unit is engaged, Ongoing Consolidation is mandatory.

Otherwise, if the unit is within 3 inches of one or more enemy units, Engaging Consolidation is mandatory.

Otherwise, if the unit is within 3 inches of one or more objectives, Objective Consolidation is mandatory.

Engaging Consolidation can force enemy units that have not yet fought this phase to be selected to fight.

Objective Consolidation must end within range of the selected objective if possible.

## Interactions

- [[Fight]] uses this procedure after a unit has completed its melee attacks.
- [[Pile-in Move]] is the matching fight-owned movement procedure that happens before attacks.
- [[Objective Control System]] is the neighbouring system that consumes the objective-positioning outcomes created by this procedure.
- [[Movement System]] remains the owner of generic movement procedures, but this move stays in Fight because it exists only as part of melee resolution.

## Restrictions

- The move is capped at 3 inches.
- The unit must follow the mandatory consolidation mode that its current state requires.
- This article does not redefine objective-control rules, generic movement rules, or any shared attack-resolution step.
- Unpublished objective-specific child concepts remain outside this page; this article only states the fight-owned movement consequences that point toward them.

# Normal Fight

## Summary

Normal Fight is the standard melee branch for an eligible engaged unit and hands attack execution to the shared attack pipeline.

## Definition

An engaged unit fights using [[Making Attacks]].

## Rules

This is the default fight type for an eligible engaged unit once it is selected to fight.

After the unit is selected through [[Fight]], it resolves its melee attacks through [[Making Attacks]].

This branch is the baseline from which fight-only exceptions, such as [[Overrun Fight]], are distinguished.

## Interactions

- [[Fight]] provides the selection and sequencing layer that determines when this branch is used.
- [[Making Attacks]] and [[Attack System]] provide the shared attack-resolution steps used once melee combat begins.
- [[Weapon System]] remains the owner of the weapon data that the selected melee attacks consume.
- [[Overrun Fight]] is the adjacent special branch for eligible units that are not resolving the standard engaged-fight case.

## Restrictions

- This article does not grant eligibility to fight by itself; that permission comes from [[Fight]].
- This article does not replace the shared attack sequence or redefine melee weapon characteristics.
- This branch applies only to the standard engaged case, not to the extra-movement exception covered by [[Overrun Fight]].

# Stealth

## Summary

Stealth is the core ability that gives a unit the benefit of cover against ranged attacks if every model in the unit has the ability.

## Definition

If every model in a unit has Stealth, the unit has the benefit of cover each time a ranged attack targets it.

## Rules

Stealth applies only when every model in the target unit has the ability.

The effect is the benefit of cover against that ranged attack.

## Interactions

- [[Benefit of Cover]] owns the cover effect granted by Stealth.
- [[Shooting System]] provides the ranged-attack context.

## Restrictions

- Stealth grants benefit of cover; it does not create Hidden or Obscuring terrain state.

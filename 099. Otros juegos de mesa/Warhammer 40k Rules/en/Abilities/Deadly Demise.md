# Deadly Demise

## Summary

Deadly Demise is the core ability that can cause a destroyed model to inflict mortal wounds on nearby units after any embarked units complete emergency disembarkation.

## Definition

Deadly Demise is a numbered ability that triggers when a model with the ability is destroyed.

## Rules

This ability takes the form Deadly Demise X.

When the model is destroyed, after any embarked units have made required emergency disembark moves, roll one D6.

On a 6, each unit within 6 inches of that model suffers X mortal wounds.

If X is random, roll separately for each affected unit.

## Interactions

- [[Emergency Disembark Move]] can resolve before Deadly Demise on a destroyed transport.
- [[Mortal Wounds]] owns the wound-loss procedure used by the ability.
- [[Destroyed]] provides the destruction timing that triggers the ability.

## Restrictions

- This article does not redefine the destruction rules or emergency disembarkation.

# Infiltrators

## Summary

Infiltrators is the core ability that allows a unit to be set up forward during deployment if every model in the unit has the ability.

## Definition

During deployment, a unit whose every model has Infiltrators can be set up forward within the stated restrictions.

## Rules

If every model in the unit has this ability, during deployment the unit can be set up anywhere on the battlefield that is more than 8 inches horizontally from the opponent's deployment zone and from all enemy units.

## Restrictions

- Infiltrators affects deployment setup, not later reserve arrival.

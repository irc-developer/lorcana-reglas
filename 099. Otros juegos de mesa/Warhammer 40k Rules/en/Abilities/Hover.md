# Hover

## Summary

Hover is the core ability that removes the normal maximum-distance penalty when a unit takes to the skies.

## Definition

Hover applies when a unit with the ability takes to the skies.

## Rules

Each time the unit takes to the skies, do not subtract 2 inches from the maximum distance.

## Interactions

- [[Flying Models]] owns the take-to-the-skies rule that Hover modifies.
- [[Movement System]] provides the wider movement context.

## Restrictions

- Hover only modifies the take-to-the-skies distance penalty; it does not create independent permission to move.

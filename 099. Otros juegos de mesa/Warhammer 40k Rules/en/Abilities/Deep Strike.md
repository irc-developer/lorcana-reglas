# Deep Strike

## Summary

Deep Strike is the core ability that lets a unit making an Ingress Move set up more flexibly from Strategic Reserves.

## Definition

Deep Strike modifies an Ingress Move for a unit whose every model has the ability.

## Rules

When the unit makes an Ingress Move, if every model in the unit has Deep Strike, it can be set up anywhere on the battlefield more than 8 inches horizontally from all enemy units.

This permission can allow setup within the opponent's deployment zone.

## Interactions

- [[Ingress Move]] is the move type modified by this ability.
- [[Strategic Reserves System]] provides the reserve-arrival framework.
- [[Deployment zone]] is relevant because Deep Strike can override the usual early-round deployment-zone restriction.

## Restrictions

- This article does not allow arrival before a unit is otherwise eligible to arrive.

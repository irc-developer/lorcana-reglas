# Command Phase

## Summary

Command Phase is the turn-phase timing container for core CP gain, command-phase battle-shock, and command-level abilities.

## Definition

Command Phase is the phase in which core CP gain, command-phase battle-shock, and command abilities are resolved.

## Rules

Command Phase resolves in this order:

1. Start of Command Phase.
2. Gain Core CP.
3. [[Command Phase Battle-shock]].
4. Command Abilities.
5. End of Command Phase.

During the active player's Command phase, battle-shock rolls are made for units that are currently battle-shocked and/or at or below half-strength.

## Interactions

- [[Turn Structure]] is the parent timing article that places this phase inside each player turn.
- [[Command Phase Battle-shock]] is the published battle-shock-owned application that occurs during this phase.
- [[Battle-shock System]] consumes this phase as the timing window for recovery and reapplication of battle-shock.
- [[Starting Strength and Half-strength]] provides the half-strength condition consumed during command-phase battle-shock.
- [[Secured Objectives]] can be affected by outcomes that depend on this phase's battle-shock timing.
- [[Using Stratagems]] consumes the command-point economy that this phase refreshes without taking ownership of the phase itself.

## Restrictions

- This article owns the phase timing, not the full rule meaning of battle-shock, stratagem use, or command-point spending.
- Gain Core CP and Command Abilities remain unresolved substeps rather than published child articles in the current source scope.
- This concept does not redefine any downstream mechanic that merely happens during the Command phase.

# Límite de unidades

## Resumen

Límite de unidades es el límite de construcción de ejército sobre cuántas copias de cada unidad pueden incluirse en un ejército.

## Definición

Límite de unidades es el número máximo de cada unidad que puede incluirse en un ejército para el tamaño de batalla elegido.

## Reglas

Ejemplos publicados:

| Tamaño de batalla | Límite de unidades |
| --- | ---: |
| Incursion | 2 |
| Strike Force | 3 |

La nota publicada indica que el límite de unidades se dobla para unidades BATTLELINE.

## Interacciones

- [[Tamaño de batalla]] determina el límite de unidades.
- [[Claves]] determina si una unidad tiene la clave BATTLELINE.
- [[Construcción de ejército]] aplica el límite de unidades antes de empezar la prebatalla.

## Restricciones

- Este artículo no define costes en puntos de unidades.
- Este artículo no sustituye restricciones de lista específicas de facción o publicación.

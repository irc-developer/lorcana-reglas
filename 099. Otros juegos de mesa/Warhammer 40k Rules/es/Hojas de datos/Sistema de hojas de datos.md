# Sistema de hojas de datos

## Resumen

El Sistema de hojas de datos organiza la información de unidad que aparece en una hoja de datos antes de que otros sistemas la consuman durante la partida.

## Alcance

Este dominio cubre el contenedor hoja de datos, su nombre, perfiles, armas, claves, composición de unidad y opciones de equipo.

## Responsabilidad del dominio

- Explicar dónde vive cada dato de una unidad.
- Separar la estructura de datos de sus efectos durante el juego.
- Enlazar hacia sistemas vecinos como habilidades, claves y armas cuando la duda ya no sea estructural.

- Pasa a [[Sistema de armas]] cuando la duda sea el comportamiento de armas o perfiles de arma durante ataques.
- Pasa a [[Sistema de claves]] cuando la duda sea cómo se interpretan las claves.

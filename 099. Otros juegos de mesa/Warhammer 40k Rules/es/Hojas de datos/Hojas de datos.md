# Hojas de datos

## Resumen

Hojas de datos define el contenedor que explica cómo funciona una unidad en batalla.

## Definición

Cada unidad tiene una hoja de datos que explica cómo funciona en batalla.

## Reglas

Una hoja de datos contiene, entre otros elementos, el nombre de la unidad, perfiles, habilidades, armas, claves, composición de unidad y opciones de equipo.

La hoja de datos indica qué información tiene la unidad; otros sistemas explican cómo se usa esa información durante la partida.

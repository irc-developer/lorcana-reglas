# Sistema de aeronaves

## Resumen

El Sistema de aeronaves reúne las reglas específicas de las unidades AERONAVE: despliegue en reservas, movimiento limitado, disparo y restricciones de carga y combate.

## Alcance

Este dominio cubre la sección 23 de las Reglas básicas.

Las reservas estratégicas, el movimiento, el disparo y el combate siguen siendo sus sistemas propietarios; este dominio solo añade las excepciones de AERONAVE.

# Sistema de acobardamiento

## Resumen

El Sistema de acobardamiento organiza las tiradas de acobardamiento, el estado acobardado y su aplicación principal en la fase de mando.

## Alcance

Este dominio cubre cuándo una unidad hace tiradas de acobardamiento, qué ocurre si las falla, cómo una unidad acobardada puede dejar de estarlo y qué sistemas consumen ese estado.

## Responsabilidad del dominio

- Mantener la regla canónica de [[Tiradas de acobardamiento]].
- Mantener el estado [[Acobardada]].
- Mantener la aplicación de [[Acobardamiento en fase de mando]].
- Enlazar a los sistemas que consumen las consecuencias: objetivos, estratagemas y acciones.

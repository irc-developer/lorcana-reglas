# Secuencia de ataque

## Resumen

La Secuencia de ataque es el flujo que determina si los dados de ataque reunidos impactan, hieren, son salvados o infligen daño.

## Definición

Siempre que se indique resolver la secuencia de ataque, el jugador activo sigue estos pasos: [[Tiradas para impactar]], [[Tiradas para herir]], [[Tiradas de salvación]] e [[Infligir daño]].

## Reglas

La secuencia tiene cuatro pasos:

1. [[Tiradas para impactar]]
2. [[Tiradas para herir]]
3. [[Tiradas de salvación]]
4. [[Infligir daño]]

En cada paso, si hay que tirar más de un D6, todas esas tiradas se hacen a la vez.

Si un ataque falla o inflige daño, la secuencia termina para ese ataque. Cuando todos los ataques hayan fallado o infligido daño, esos ataques se consideran resueltos.

## Impactos críticos y heridas críticas

Los impactos críticos siguen siendo impactos. Las heridas críticas siguen siendo heridas.

Además, otras reglas pueden activarse por impactos críticos o heridas críticas, como [IMPACTOS LETALES] y [HERIDAS DEVASTADORAS].

## Interacciones

- [[Resolver ataques]] reúne los dados que entran en esta secuencia.
- [[Heridas mortales]] puede aplicarse junto a daño normal, pero tiene su propia regla de resolución.
- [[Sistema de armas]] conserva habilidades de arma que pueden modificar esta secuencia.

## Restricciones

- Esta secuencia no elige armas ni blancos; eso ya debe estar fijado.
- Esta secuencia no autoriza a una unidad a disparar o combatir.
- Algunas reglas pueden saltar o modificar pasos concretos, pero solo si lo indican expresamente.

## Navegación

- Sigue con [[Tiradas para impactar]] para el primer paso.
- Sigue con [[Tiradas para herir]] cuando ya se hayan determinado los impactos.
- Sigue con [[Tiradas de salvación]] cuando ya se hayan determinado las heridas.
- Sigue con [[Infligir daño]] para aplicar los ataques no salvados.

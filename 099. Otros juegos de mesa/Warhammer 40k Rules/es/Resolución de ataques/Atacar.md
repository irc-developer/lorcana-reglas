# Atacar

## Resumen

Atacar es el procedimiento común que se usa cuando una unidad dispara o combate: primero se eligen armas, luego blancos y finalmente se resuelven los ataques.

## Definición

Siempre que una unidad dispara o combate, el jugador activo sigue estos pasos: [[Elegir armas]], [[Elegir blancos]] y [[Resolver ataques]].

## Reglas

Al disparar, las miniaturas realizan ataques a distancia con armas a distancia.

Al combatir, las miniaturas realizan ataques de combate con armas de combate.

El procedimiento común no concede por sí mismo permiso para disparar o combatir. Solo estructura cómo se resuelven los ataques cuando una unidad ya ha sido autorizada por una fase, regla o habilidad.

## Interacciones

- [[Elegir armas]] determina con qué armas atacará cada miniatura.
- [[Elegir blancos]] fija las unidades enemigas que serán blanco de cada arma.
- [[Resolver ataques]] procesa las armas y dados contra cada unidad blanco.
- [[Secuencia de ataque]] se aplica una vez reunidos los dados de ataque.
- [[Sistema de disparo]] y [[Sistema de combate]] consumen este procedimiento desde sus propias fases.

## Restricciones

- Una miniatura sin armas a distancia no puede realizar ataques a distancia.
- Una miniatura sin armas de combate no puede realizar ataques de combate.
- Al disparar o combatir puedes elegir blancos diferentes para cada arma, pero cada blanco debe ser legal.
- Si no puedes elegir blanco para un arma, o decides no elegir blanco para un arma a distancia, la miniatura no ataca con esa arma.

# Tiradas para impactar

## Resumen

Las Tiradas para impactar determinan qué dados de ataque se convierten en impactos y cuáles fallan.

## Definición

Haz una tirada para impactar por cada dado de ataque tirando un D6.

## Reglas

Para cada resultado, comprueba la primera condición aplicable:

- Un resultado sin modificar de 1 falla.
- Un resultado sin modificar de 6 es un impacto crítico.
- Un resultado igual o mayor que el atributo HP/HA del ataque es un impacto.
- Cualquier otro resultado falla.

Los impactos continúan a [[Tiradas para herir]].

## Interacciones

- Un impacto crítico sigue siendo un impacto.
- Algunas habilidades se activan con impactos críticos.
- Modificadores y repeticiones pueden afectar a tiradas para impactar si una regla lo permite.

## Restricciones

- Solo los ataques que impactan pasan al paso de herir.
- Un ataque que falla en este paso termina su secuencia.
- Esta regla no define cuántos dados se tiran; eso lo determina [[Resolver ataques]].

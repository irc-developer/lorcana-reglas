# Acciones

## Resumen

Las acciones son tareas de campo de batalla definidas por misiones o reglas que una unidad puede iniciar, realizar y completar para producir un efecto.

## Definición

Las acciones representan tareas vitales que tus fuerzas deben llevar a cabo, como buscar arqueotecnología perdida o completar un ritual.

## Reglas

Una acción indica cuándo comienza, qué unidades pueden realizarla, cuántas veces puede iniciarse, cuándo se completa y qué efecto tiene al completarse.

Una unidad que inicia una acción queda limitada hasta el final del turno: no es apta para disparar, salvo unidades TITÁNICA, y no es apta para declarar cargas.

Una acción iniciada no se completa si la unidad que la realiza mueve, salvo al agruparse o consolidar, o si abandona el campo de batalla.

## Interacciones

- [[Realizar acciones]] define la estructura común de una acción.
- [[Iniciar una acción]] define los requisitos para empezar una acción.
- [[Completar una acción]] define cuándo una acción iniciada falla antes de completarse.
- [[Tiradas de acobardamiento]] impide iniciar acciones y puede impedir completar una acción ya iniciada.
- [[Sistema de movimiento]] determina qué movimientos afectan a la finalización.
- [[Sistema de disparo]] y [[Sistema de carga]] consumen las restricciones posteriores a iniciar una acción.

## Restricciones

- Este artículo no define los efectos de una acción concreta.
- Las acciones usables pueden estar en packs de misión u otras publicaciones.
- Que una unidad sea apta para una acción no implica que controle un objetivo.

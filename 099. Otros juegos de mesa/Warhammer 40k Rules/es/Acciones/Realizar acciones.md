# Realizar acciones

## Resumen

Realizar acciones es el procedimiento general por el que una regla permite a una unidad iniciar una tarea y resolver su efecto si llega a completarla.

## Definición

Algunas reglas permiten que las unidades realicen acciones. Cada acción especifica sus campos de inicio, unidades, límite de usos, finalización, efecto y restricciones adicionales.

## Reglas

Cada acción especifica:

- **COMIENZA**: cuándo se inicia.
- **UNIDADES**: qué unidades pueden realizarla.
- **LÍMITE DE USOS**: cuántas veces pueden iniciarla las unidades amigas.
- **SE COMPLETA**: cuándo se completa.
- **EFECTO**: qué efectos tiene completarla.
- Cualquier restricción adicional aplicable.

El efecto de la acción se activa cuando la acción se completa.

## Interacciones

- [[Iniciar una acción]] comprueba si una unidad puede empezar la acción.
- [[Completar una acción]] comprueba si la acción llega al punto en que se activa su efecto.
- [[Acciones]] mantiene el concepto general y la navegación del dominio.

## Restricciones

- El procedimiento no permite usar una acción cuyo propio texto no autorice su inicio.
- Una acción de ejemplo no es una acción jugable salvo que otra regla lo indique.

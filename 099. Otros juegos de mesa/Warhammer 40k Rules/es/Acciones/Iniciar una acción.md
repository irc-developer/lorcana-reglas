# Iniciar una acción

## Resumen

Iniciar una acción es el control de aptitud que determina si una unidad puede empezar una acción en el momento indicado por esa acción.

## Definición

Una unidad es apta para iniciar una acción si no se le aplica ninguna de las condiciones que la regla enumera como impedimentos.

## Reglas

Una unidad no puede iniciar una acción si se cumple cualquiera de estas condiciones:

- No está en el campo de batalla.
- Es una unidad AERONAVE/FORTIFICACIÓN.
- Está acobardada.
- Tiene un atributo [[Control de objetivos (CO)|CO]] de 0 o “-”.
- Está trabada, salvo si es TITÁNICA.
- Avanzó o retrocedió este turno.
- Inició otra acción este turno.

Si una unidad inicia una acción, hasta el final del turno:

- No es apta para disparar, salvo unidades TITÁNICA.
- No es apta para declarar cargas.

## Interacciones

- [[Tiradas de acobardamiento]] puede hacer que una unidad esté acobardada y no pueda iniciar acciones.
- [[Control de objetivos (CO)]] importa porque CO 0 o “-” impide iniciar acciones.
- [[Avanzar]] y [[Retroceder]] impiden iniciar acciones durante ese turno.
- [[Sistema de disparo]] y [[Sistema de carga]] aplican las restricciones posteriores a iniciar la acción.

## Restricciones

- La unidad debe cumplir también las restricciones propias de la acción concreta.
- Iniciar una acción no garantiza completarla.
- Una unidad solo puede iniciar una acción por turno.

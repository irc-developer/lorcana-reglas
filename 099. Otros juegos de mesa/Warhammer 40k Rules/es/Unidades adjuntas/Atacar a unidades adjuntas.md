# Atacar a unidades adjuntas

## Resumen

Atacar a unidades adjuntas define qué atributo de Resistencia se usa y cuándo se considera destruida una unidad adjunta.

## Reglas

Siempre que un ataque tome como blanco una unidad adjunta, si esa unidad incluye miniaturas guardaespaldas, se usa el atributo de Resistencia más alto de esas miniaturas guardaespaldas al resolver el ataque.

Esto se aplica aunque una unidad líder o apoyo de esa unidad adjunta tenga un atributo de Resistencia diferente.

Si la unidad adjunta solo incluye miniaturas líder o apoyo, se usa el atributo de Resistencia más alto entre esas miniaturas.

Las reglas que se activan cuando una unidad es eliminada solo se activan cuando se elimina la última miniatura que empezó la batalla en esa unidad adjunta.

## Interacciones

- [[Tiradas para herir]] usa la Resistencia determinada por esta regla.
- [[Secuencia de ataque]] sigue resolviéndose con normalidad.
- [[Unidad adjunta]] define la entidad que está siendo atacada.

# Sistema de control de objetivos

## Resumen

El Sistema de control de objetivos es el artículo de sistema del dominio de Objetivos. Organiza cómo se representan los objetivos, cómo se calcula su control y cómo puede persistir ese control.

## Alcance

Este dominio cubre objetivos de terreno, marcadores de objetivo, nivel de control, objetivos asegurados y el atributo Control de Objetivos.

No cubre el dominio completo de [[Sistema de acciones|acciones]], que se mantiene como carpeta propia.

## Responsabilidad del dominio

- Objetivos mantiene la representación de objetivos, el cálculo de control, los objetivos asegurados y los términos de apoyo.
- [[Sistema de terreno y visibilidad]] mantiene las áreas de terreno que pueden usarse como objetivos.
- [[Sistema de movimiento]] determina cómo las miniaturas entran o salen del alcance de un objetivo.
- [[Sistema de acobardamiento]] puede modificar el atributo CO, pero no absorbe la responsabilidad de las reglas de objetivos.

## Navegación

- Empieza por [[Objetivos de terreno]] para la representación normal de objetivos mediante áreas de terreno.
- Lee [[Objetivos fuera de un área de terreno]] si el objetivo se representa con marcador.
- Lee [[Nivel de control]] para determinar quién controla un objetivo.
- Lee [[Asegurar objetivos]] cuando una regla mantiene el control aunque ya no haya unidades dentro del alcance.
- Lee [[Control de objetivos (CO)]] para el atributo que se suma al calcular el control.

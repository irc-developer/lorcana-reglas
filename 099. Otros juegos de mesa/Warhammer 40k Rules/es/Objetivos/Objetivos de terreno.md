# Objetivos de terreno

## Resumen

Objetivos de terreno es la regla que representa un objetivo mediante un área de terreno cuando la ubicación indicada por la misión coincide con una de ellas.

## Definición

Si una misión usa objetivos, la ubicación de cada objetivo debería coincidir con un [[Área de terreno|área de terreno]]; esa área de terreno es el objetivo y se denomina [[Objetivo de terreno]].

## Reglas

Cuando un objetivo se representa mediante terreno, el objetivo es el área de terreno, no un marcador separado.

Al medir distancias hacia y desde un objetivo, se mide hacia y desde la parte más cercana de este.

Si la ubicación del objetivo no coincide con un área de terreno, usa [[Objetivos fuera de un área de terreno]].

## Interacciones

- [[Área de terreno]] es el estado de terreno que se usa como objetivo.
- [[Nivel de control]] usa esta representación para saber qué miniaturas están dentro del alcance.
- [[Sistema de terreno y visibilidad]] sigue siendo responsable del terreno aunque el área funcione como objetivo.

## Restricciones

- Este artículo no define cómo se coloca el terreno.
- Este artículo no calcula quién controla el objetivo; eso lo hace [[Nivel de control]].

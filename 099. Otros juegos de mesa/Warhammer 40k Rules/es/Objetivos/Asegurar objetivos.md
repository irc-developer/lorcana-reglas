# Asegurar objetivos

## Resumen

Asegurar objetivos es la regla de persistencia que mantiene un objetivo bajo el control de un jugador aunque ya no tenga unidades dentro de su alcance.

## Definición

Algunas reglas permiten que un objetivo sea asegurado por el ejército de un jugador.

## Reglas

Cuando un objetivo está asegurado por el ejército de un jugador, permanece bajo su control aunque ese jugador ya no tenga unidades dentro de su alcance.

Ese control persiste hasta que el nivel de control del oponente sobre ese objetivo sea mayor que el suyo al final de una fase.

## Interacciones

- [[Nivel de control]] determina cuándo el oponente supera el control del jugador que aseguró el objetivo.
- [[Control de objetivos (CO)]] sigue importando para intentar arrebatar el objetivo.
- Las habilidades de hojas de datos pueden asegurar objetivos, pero esas habilidades pertenecen a [[Sistema de hojas de datos]].

## Restricciones

- Este artículo no define qué reglas aseguran objetivos; solo define qué significa que un objetivo esté asegurado.
- El oponente debe tener un nivel de control mayor, no igual.

# Nivel de control

## Resumen

Nivel de control es el cálculo que suma el atributo CO de las miniaturas de cada jugador dentro del alcance de un objetivo para determinar quién lo controla.

## Definición

Al final de cada fase y turno, se suma el atributo CO de todas las miniaturas de cada ejército que estén dentro del alcance de un objetivo.

## Reglas

Al inicio de la batalla, ningún objetivo está controlado por ningún jugador.

Para obtener el control de un objetivo, un jugador necesita tener una o más miniaturas con un atributo [[Control de objetivos (CO)|CO]] de 1 o más dentro de su alcance.

Al final de cada fase y turno:

- El jugador con el mayor nivel de control sobre ese objetivo lo controla.
- Si ambos jugadores tienen el mismo nivel de control, el objetivo no está controlado por ninguno, salvo que esté asegurado.

Mientras una o más unidades de un ejército estén dentro del alcance de un objetivo que ese jugador controla, cada una de esas unidades que contenga una o más miniaturas con CO de 1 o más controla ese objetivo.

## Interacciones

- [[Objetivos de terreno]] define el alcance mediante el área de terreno.
- [[Objetivos fuera de un área de terreno]] define el alcance mediante el marcador.
- [[Asegurar objetivos]] modifica qué ocurre cuando hay empate o cuando una unidad abandona el alcance.
- [[Tiradas de acobardamiento]] puede reducir el CO de una unidad acobardada a “-”, lo que afecta a este cálculo.

## Restricciones

- El control se determina al final de cada fase y turno.
- Empatar el nivel de control no arrebata un objetivo asegurado.
- Solo cuentan miniaturas dentro del alcance del objetivo.

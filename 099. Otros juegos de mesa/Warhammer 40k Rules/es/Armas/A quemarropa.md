# A quemarropa

## Resumen

A quemarropa es una habilidad de arma de las Reglas básicas.

## Reglas

Las unidades que incluyan una o más miniaturas con armas [A QUEMARROPA] pueden disparar usando disparo a quemarropa. Al usar otros tipos de disparo, una miniatura no MONSTRUO/VEHÍCULO debe elegir entre sus armas [A QUEMARROPA] o sus otras armas a distancia.

## Interacciones

- [[Habilidades de armas]] contiene el marco común de habilidades entre corchetes.
- [[Sistema de ataque]] usa esta regla al resolver ataques con esa arma.

## Restricciones

- Esta habilidad solo se aplica a ataques realizados con armas que tengan la habilidad correspondiente.
- Si la habilidad incluye número, clave o condición, usa el texto exacto del perfil de arma.

## Referencias

- Reglas básicas oficiales: A quemarropa 24.07.

# Sistema de armas

## Resumen

El Sistema de armas organiza perfiles de armas, atributos de armas y habilidades de armas.

## Alcance

Este dominio cubre cómo se describen las armas en las hojas de datos y cómo sus atributos y habilidades alimentan la secuencia de ataque.

El procedimiento de resolver ataques pertenece a [[Sistema de ataque]], pero las reglas de armas explican qué datos usa ese procedimiento.

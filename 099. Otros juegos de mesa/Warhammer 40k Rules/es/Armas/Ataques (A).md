# Ataques (A)

## Resumen

Ataques (A) es un atributo de arma.

## Definición

Ataques (A) indica cuántos dados de ataque se emplean cada vez que se usa esa arma.

## Interacciones

- [[Atributos de armas]] lista el resto de atributos del perfil.
- [[Sistema de ataque]] usa este valor al resolver ataques cuando corresponde.

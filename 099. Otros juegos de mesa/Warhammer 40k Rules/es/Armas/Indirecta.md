# Indirecta

## Resumen

Indirecta es una habilidad de arma de las Reglas básicas.

## Reglas

Las unidades con armas [INDIRECTA] pueden disparar usando disparo indirecto.

## Interacciones

- [[Habilidades de armas]] contiene el marco común de habilidades entre corchetes.
- [[Sistema de ataque]] usa esta regla al resolver ataques con esa arma.

## Restricciones

- Esta habilidad solo se aplica a ataques realizados con armas que tengan la habilidad correspondiente.
- Si la habilidad incluye número, clave o condición, usa el texto exacto del perfil de arma.

- Reglas básicas oficiales: Indirecta 24.19.

# Perfil de arma

## Resumen

Un perfil de arma reúne los atributos y habilidades que se usan cuando una miniatura ataca con esa arma.

## Reglas

El perfil de arma contiene los valores necesarios para determinar alcance, cantidad de dados de ataque, habilidad usada para impactar, fuerza, penetración y daño.

Las habilidades de armas se muestran normalmente entre corchetes junto al nombre o perfil del arma.

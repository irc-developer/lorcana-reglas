# No hay dolor

## Resumen

No hay dolor es una habilidad básica de la sección de referencia de Warhammer 40,000.

## Reglas

No hay dolor X+ permite tirar 1D6 cada vez que una miniatura con esta habilidad vaya a perder una herida. Con X+, esa herida no se pierde.

# Sistema de movimiento

## Resumen

El Sistema de movimiento es el mapa del dominio que reúne las reglas que permiten a una unidad cambiar de posición, permanecer inmóvil o aplicar excepciones de terreno, claves y habilidades.

## Alcance

Este dominio cubre el procedimiento base de [[Mover unidades]], los tipos principales de movimiento de la fase de movimiento, los límites de distancia, las condiciones de finalización y las excepciones publicadas para miniaturas MONSTRUO, VEHÍCULO y unidades que pueden VOLAR.

También mantiene los movimientos especiales localizados que pertenecen al dominio, como [[Movimiento de avanzadilla]] y [[Movimiento súbito]].

## Responsabilidad del dominio

- Mantener las reglas de movimiento que indican cómo se desplazan miniaturas y unidades.
- Separar el procedimiento base de los tipos concretos de movimiento.
- Indicar cuándo una excepción modifica el movimiento sin cambiar el tipo de movimiento.
- Mantener enlaces hacia sistemas vecinos cuando el caso pertenece a otra carpeta.

## Navegación

- Empieza por [[Mover unidades]] para el procedimiento común.
- Usa [[Permanecer inmóvil]], [[Movimiento normal]], [[Avanzar]] o [[Retroceder]] para los tipos principales de la fase de movimiento.
- Usa [[Atributo M]] y [[Atributos Movimiento diferentes]] para dudas sobre distancia máxima.
- Usa [[Mover en vertical]] y [[Desplegar o terminar un movimiento]] cuando el problema sea el terreno.
- Usa [[Mover monstruos y vehículos]] o [[Miniaturas que vuelan]] cuando una clave permita atravesar miniaturas o terreno de forma especial.
- Usa [[Movimiento de avanzadilla]] para reposicionamiento prebatalla y [[Movimiento súbito]] para movimiento activado por una regla.

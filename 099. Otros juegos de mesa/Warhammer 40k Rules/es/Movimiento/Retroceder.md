# Retroceder

## Resumen

Retroceder es el tipo de movimiento que usa una unidad trabada para dejar de estarlo, resolviendo Retirada ordenada o Huida desesperada.

## Definición

Una unidad trabada puede retroceder moviendo hasta su atributo M para terminar no trabada.

## Reglas

La distancia máxima es el [[Atributo M|atributo M]] de la unidad.

La unidad debe estar trabada.

Antes de mover, elige un modo de retroceso:

- **Retirada ordenada**: puedes elegir este modo si tu unidad no está acobardada.
- **Huida desesperada**: si no puedes o no eliges Retirada ordenada, elige este modo y haz una [[Tiradas de riesgo|tirada de riesgo]] por cada miniatura de la unidad.

Durante Huida desesperada, cada miniatura que se mueva puede atravesar miniaturas enemigas.

Después de mover:

- La unidad debe no estar trabada.
- Hasta el final del turno, salvo que una regla indique lo contrario, no es apta para disparar, declarar cargas ni iniciar acciones.
- Si usó Huida desesperada y la unidad no está acobardada, debes hacer una [[Tiradas de acobardamiento|tirada de acobardamiento]] para esa unidad.

## Interacciones

- [[Mover unidades]] contiene el procedimiento base y la comprobación final.
- [[Miniaturas que vuelan]] puede aplicarse a unidades VOLADORAS al hacer un movimiento de retroceder.
- [[Fase de disparo]], [[Fase de carga]] y [[Acciones]] consumen las restricciones posteriores.

## Restricciones

- Solo puede elegirse si la unidad está trabada.
- La unidad debe terminar no trabada.
- Retroceder no permite disparar, cargar ni iniciar acciones ese turno salvo excepción específica.

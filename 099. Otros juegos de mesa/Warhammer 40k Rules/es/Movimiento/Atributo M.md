# Atributo M

## Resumen

El atributo M es el valor de Movimiento de una miniatura y se usa como distancia máxima cuando una regla indica que una unidad mueve usando ese atributo.

## Definición

El atributo M representa la velocidad a la que una miniatura se desplaza por el campo de batalla.

## Interacciones

- [[Movimiento normal]] y [[Retroceder]] usan el atributo M como distancia máxima.
- [[Avanzar]] usa el atributo M más una tirada de avanzar.
- [[Atributos Movimiento diferentes]] explica qué ocurre cuando las miniaturas de una misma unidad no comparten el mismo atributo M.
- [[Mover unidades]] contiene el procedimiento general que consume este valor.

## Restricciones

- El atributo M no sustituye los requisitos de coherencia, trabado, terreno o finalización del movimiento.
- Si una miniatura tiene un atributo M de “-”, puede desplegarse en el campo de batalla, pero no puede moverse salvo que una regla lo permita.

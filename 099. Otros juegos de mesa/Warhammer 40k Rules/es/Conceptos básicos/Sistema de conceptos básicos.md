# Sistema de conceptos básicos

## Resumen

El Sistema de conceptos básicos es el punto de entrada para las reglas fundacionales que otros sistemas consumen antes de aplicar procedimientos de fase, excepciones o reglas específicas.

## Alcance

Este dominio cubre la estructura básica de ejército, unidades y miniaturas; las tiradas compartidas; la medición; el despliegue; la coherencia; el trabado; y los estados básicos de efectivos, eliminación y retorno de miniaturas.

## Responsabilidad del dominio

- Mantener los conceptos fundacionales que otros sistemas usan.
- Separar las definiciones generales de los procedimientos de fase.
- Servir como mapa de navegación para las reglas básicas.

## Navegación

- Empieza aquí cuando la duda sea sobre estructura o estado básico, no sobre una fase concreta.
- Lee [[Unidades y miniaturas]] para la diferencia entre unidad y miniatura.
- Lee [[Dados]] para la notación común de D6, D3 y resultados objetivo.
- Lee [[Tiradas de liderazgo]] para la familia de tiradas que consumen otras reglas.
- Lee [[Tiradas de acobardamiento]] cuando la duda sea cómo una unidad pasa a estar acobardada.
- Pasa a [[La ronda de batalla]] o [[Fase de mando]] cuando la duda sea de secuencia de juego.

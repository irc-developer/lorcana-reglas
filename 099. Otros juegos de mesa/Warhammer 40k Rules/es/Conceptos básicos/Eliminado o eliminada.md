# Eliminado o eliminada

## Resumen

Eliminado o eliminada define cuándo una miniatura o unidad deja de estar disponible para usar habilidades, ser elegida o ser tomada como blanco.

## Definición

Una miniatura resulta eliminada normalmente cuando sus heridas se reducen a 0 o menos. Cuando todas las miniaturas de una unidad resultan eliminadas, esa unidad resulta eliminada.

## Reglas

Cuando una miniatura resulta eliminada, primero resuelve las reglas que se activan cuando resulta eliminada y después retírala del campo de batalla.

Si la miniatura resultó eliminada por un ataque, salvo que se indique lo contrario, esas reglas solo se resuelven y la miniatura solo se retira después de resolver los ataques de la unidad atacante.

Salvo que se indique lo contrario, las miniaturas y unidades eliminadas no pueden usar habilidades ni ser elegidas o tomadas como blanco por reglas.

## Interacciones

- [[Infligir daño]] es una de las formas habituales de hacer que una miniatura resulte eliminada.
- [[Revivir]] define cómo miniaturas eliminadas pueden volver a una unidad.

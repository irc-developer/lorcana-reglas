# Tiradas de liderazgo

## Resumen

Las Tiradas de liderazgo son tiradas de 2D6 usadas para comprobar si una unidad supera una prueba indicada por otra regla.

## Definición

Para hacer una tirada de liderazgo de una unidad, el jugador que la controla tira 2D6.

La tirada tiene éxito si el resultado es igual o mayor que alguno de los atributos L de esa unidad. En caso contrario, falla.

## Reglas

La regla que indicó hacer la tirada de liderazgo describe qué ocurre si la tirada tiene éxito o falla.

Una tirada de liderazgo es una familia de tirada fundacional; no define por sí misma el efecto que se aplica.

## Interacciones

- [[Dados]] define la notación 2D6 usada por esta tirada.
- [[Tiradas de acobardamiento]] usa una tirada de liderazgo para determinar si una unidad pasa a estar acobardada o deja de estarlo cuando una regla lo permite.

## Restricciones

- Este artículo no define el estado acobardado.
- Este artículo no define cuándo se hacen tiradas de acobardamiento en la fase de mando.

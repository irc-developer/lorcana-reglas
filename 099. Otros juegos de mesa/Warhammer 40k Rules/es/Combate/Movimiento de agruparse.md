# Movimiento de agruparse

## Resumen

Movimiento de agruparse es el movimiento de combate que permite a una unidad apta mover hasta 3" para acercarse y trabarse con sus blancos de agruparse.

## Definición

En la fase de combate, una unidad apta puede mover hasta 3" siguiendo las restricciones de agruparse.

## Reglas

La distancia máxima es 3".

Una unidad es apta si es la fase de combate y se aplica alguna de estas condiciones:

- Está trabada.
- Hizo un [[Movimiento de carga]] este turno.
- Se eligió para librar un [[Combate a la carrera]] en esta fase.

Antes de mover, elige blancos de agruparse:

- Si la unidad está trabada, elige todas las unidades enemigas con las que esté trabada.
- Si no lo está, elige una o más unidades enemigas a 5" o menos.

Al mover:

- Las miniaturas en contacto peana con peana con alguna miniatura enemiga no pueden mover.
- Cada miniatura que mueva debe acabar más cerca del blanco de agruparse más cercano y, si es posible, trabada con él.

Después de mover:

- La unidad debe estar trabada.
- Cada miniatura que empezó el movimiento trabada con una unidad enemiga debe seguir trabada con esa unidad enemiga.

## Interacciones

- [[Combatir]] usa este movimiento antes de resolver ataques de combate.
- [[Combate a la carrera]] puede añadir un movimiento de agruparse adicional.
- [[Movimiento de consolidar]] es el movimiento equivalente que ocurre después de combatir.
- [[Sistema de movimiento]] conserva las reglas generales de movimiento.

## Restricciones

- El movimiento está limitado a 3".
- Las miniaturas en contacto peana con peana no pueden mover.
- La unidad debe terminar trabada.
- No redefine movimientos normales, avanzar, retroceder ni cargar.

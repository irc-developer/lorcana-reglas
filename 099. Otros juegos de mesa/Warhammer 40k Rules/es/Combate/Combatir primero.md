# Combatir primero

## Resumen

Combatir primero es el estado que hace que una unidad se resuelva en la ventana inicial de combate antes del resto de unidades aptas.

## Definición

Una unidad es una [[Unidad Combatir primero]] mientras todas sus miniaturas tienen la habilidad Combatir primero.

## Reglas

En el paso de combatir, las unidades con Combatir primero aptas para combatir se eligen antes que el resto de unidades aptas.

Los jugadores alternan la elección de unidades Combatir primero empezando por el jugador cuyo turno está en curso.

Si después de resolver un combate del resto de combates aparece una unidad Combatir primero apta para combatir, se vuelve a la ventana de unidades Combatir primero.

## Interacciones

- [[Combatir]] consume este estado para ordenar la elección de unidades.
- [[Unidad Combatir primero]] define el estado de unidad.
- [[Movimiento de carga]] concede esta habilidad a las miniaturas de la unidad que carga hasta el final del turno.
- [[Contraofensiva]] puede alterar la prioridad de combate desde el dominio de estratagemas.

## Restricciones

- Combatir primero cambia el orden de selección, no la secuencia de ataque.
- No hace que una unidad no apta pase a ser apta para combatir por sí mismo.
- Este artículo no define las reglas que conceden la habilidad salvo las interacciones publicadas.

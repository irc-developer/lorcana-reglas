# Combatir

## Resumen

Combatir es el paso en el que los jugadores alternan la elección de unidades aptas para resolver combates, primero con unidades Combatir primero y después con el resto.

## Definición

Los jugadores alternan la elección de unidades aptas para combatir, resolviendo primero los combates de unidades con [[Combatir primero]] y después el resto de combates.

## Reglas

Una unidad es apta para combatir si no se ha elegido para combatir previamente en esta fase y se cumple una de estas condiciones:

- Está trabada, o estaba trabada al inicio del paso de combatir.
- Hizo un [[Movimiento de carga]] este turno.

Primero se resuelven los combates de unidades con [[Combatir primero]]. Empezando por el jugador cuyo turno está en curso, los jugadores se alternan para elegir una unidad [[Unidad Combatir primero|Combatir primero]] amiga apta para combatir.

Después se resuelve el resto de combates. Empezando por el jugador que acaba de pasar a este paso, los jugadores se alternan para elegir una unidad amiga apta para combatir.

Después de resolver un combate en el paso de resolver el resto de combates, si hay alguna unidad con [[Combatir primero]] que ahora sea apta para combatir, se vuelve al paso de resolver combates de unidades con Combatir primero.

Cuando una unidad es elegida para combatir, se elige un tipo de combate para el que sea apta:

- [[Combate normal]]
- [[Combate a la carrera]]

## Interacciones

- [[Movimiento de agruparse]] ocurre antes de elegir y resolver combates.
- [[Sistema de ataque]] resuelve los ataques cuando una unidad combate como se describe en [[Atacar]].
- [[Movimiento de carga]] suele conceder [[Combatir primero]] hasta el final del turno.
- [[Movimiento de consolidar]] ocurre después de la resolución de combates.

## Restricciones

- Una unidad no puede elegirse para combatir más de una vez por fase salvo que otra regla lo permita.
- Este artículo no redefine la secuencia de ataque ni las reglas de armas de combate.
- La aptitud para combatir no garantiza que todos los modelos puedan atacar.

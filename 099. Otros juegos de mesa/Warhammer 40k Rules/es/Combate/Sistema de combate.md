# Sistema de combate

## Resumen

El Sistema de combate reúne las reglas de la fase de combate: agruparse, elegir unidades para combatir, resolver ataques de combate y consolidar.

## Alcance

Este dominio cubre [[Movimiento de agruparse]], [[Combatir]], [[Combate normal]], [[Combate a la carrera]], [[Movimiento de consolidar]], [[Combatir primero]] y [[Unidad Combatir primero]].

## Responsabilidad del dominio

Combate decide:

- qué unidades son aptas para agruparse;
- cómo se ordenan los combates;
- cómo funciona [[Combatir primero]];
- qué tipo de combate puede resolver una unidad;
- cuándo se hace un [[Movimiento de consolidar]].

Combate no decide:

- cómo se resuelven internamente las tiradas de ataque;
- cómo se declaran cargas;
- cómo funcionan perfiles de armas o habilidades de arma.

## Navegación

- Empieza por [[Fase de combate]] si la duda es cuándo ocurre la fase.
- Usa [[Movimiento de agruparse]] para la primera parte de la fase.
- Usa [[Combatir]] para elegir unidades y ordenar combates.
- Usa [[Combate normal]] o [[Combate a la carrera]] para el tipo de combate elegido.
- Usa [[Movimiento de consolidar]] después de resolver combates.
- Pasa a [[Sistema de ataque]] cuando una unidad ya está haciendo ataques de combate.
- Pasa a [[Sistema de carga]] si la duda nace de una carga previa.

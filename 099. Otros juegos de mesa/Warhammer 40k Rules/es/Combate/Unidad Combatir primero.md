# Unidad Combatir primero

## Resumen

Unidad Combatir primero es una unidad cuyas miniaturas tienen la habilidad Combatir primero y que, si es apta para combatir, se elige en la ventana prioritaria de combate.

## Definición

Una unidad es Combatir primero mientras todas sus miniaturas tienen [[Combatir primero]].

## Reglas

Una unidad Combatir primero apta para combatir se resuelve en el paso de resolver combates de unidades con Combatir primero.

El estado se comprueba cuando la unidad debe ser elegida para combatir.

## Interacciones

- [[Combatir]] usa este estado para ordenar la elección de unidades.
- [[Combatir primero]] define la habilidad/estado que crea esta categoría.
- [[Movimiento de carga]] puede hacer que una unidad sea Combatir primero hasta el final del turno.

## Restricciones

- La unidad debe seguir siendo apta para combatir.
- No cambia por sí misma qué miniaturas pueden atacar ni cómo se resuelve el daño.

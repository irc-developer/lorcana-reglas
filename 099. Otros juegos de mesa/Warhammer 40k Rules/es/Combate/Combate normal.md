# Combate normal

## Resumen

Combate normal es el tipo de combate básico para una unidad trabada que ha sido elegida para combatir.

## Definición

Una unidad trabada combate como se describe en [[Atacar]].

## Reglas

La unidad debe estar trabada.

Cuando se elige este tipo de combate, la unidad combate como se describe en [[Atacar]].

[[Combate a la carrera]] es la rama especial para una unidad apta que no está resolviendo el caso trabado estándar.

## Interacciones

- [[Combatir]] decide cuándo se elige la unidad y qué tipo de combate puede resolver.
- [[Sistema de ataque]] resuelve las tiradas y el daño una vez iniciados los ataques de combate.
- [[Sistema de armas]] conserva los perfiles y habilidades de armas de combate.

## Restricciones

- No concede por sí mismo aptitud para combatir.
- No redefine ataques, armas de combate ni secuencia de ataque.
- Aplica al caso estándar de unidad trabada.

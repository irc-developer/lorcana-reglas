# Completamente visible

## Resumen

Completamente visible indica que toda la parte relevante de una miniatura que mira hacia la observadora es visible para ella.

## Definición

Una miniatura es completamente visible si toda la parte de esa miniatura que mira hacia la miniatura observadora es visible para la miniatura observadora.

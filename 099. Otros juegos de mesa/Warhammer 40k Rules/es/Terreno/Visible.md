# Visible

## Resumen

Visible es el término para una miniatura o unidad que puede ser vista al menos en parte por una miniatura observadora.

## Definición

Una miniatura es visible si cualquier parte de ella es visible para la miniatura observadora, después de aplicar línea de visión y restricciones de terreno.

# Colocar terreno

## Resumen

Colocar terreno define cómo los elementos de terreno entran en el campo de batalla y cómo pueden formar áreas de terreno.

## Definición

Antes de la batalla, los jugadores colocan elementos de terreno en el campo de batalla mediante límites definidos, elementos individuales o grupos de elementos que delimitan un área.

## Reglas

El terreno puede colocarse de estas maneras:

- Definir los límites de una ubicación, por ejemplo con una peana o tapete, y colocar elementos de terreno completamente dentro.
- Colocar un único elemento de terreno directamente en el campo de batalla.
- Colocar dos o más elementos de terreno directamente en el campo de batalla de forma que juntos definan los límites de un área.

En cada caso, el área ocupada por esos límites o elementos se denomina [[Área de terreno]].

El mapa de despliegue de una misión puede definir la ubicación y dimensiones de cada área de terreno. Si no lo hace, los jugadores deben acordarlo antes de la batalla.

## Interacciones

- [[Área de terreno]] es el estado de campo de batalla que resulta de esta regla.
- [[Categorías de terreno]] clasifica los elementos colocados.
- [[Sistema de control de objetivos]] puede usar áreas de terreno como objetivos de terreno.

## Restricciones

- Este artículo no decide qué categoría tiene cada elemento de terreno.
- Si la misión define requisitos de terreno, esos requisitos tienen prioridad sobre un acuerdo libre entre jugadores.

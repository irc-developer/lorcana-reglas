# Categorías de terreno

## Resumen

Categorías de terreno clasifica los elementos de terreno para determinar cómo afectan al movimiento y la visibilidad.

## Definición

Cada elemento de terreno pertenece a una categoría de terreno. Las categorías oficiales de esta sección son descubierto, ligero y denso.

## Reglas

Las categorías publicadas son:

- [[Descubierto]]: ofrece poca protección y puede atravesarse sin problema.
- [[Ligero]]: puede ofrecer cobertura sin frenar de forma significativa el avance.
- Denso: obstaculiza incluso a máquinas de guerra grandes y puede ocultar unidades completas.

Un área de terreno puede contener elementos de terreno de diferentes categorías.

## Interacciones

- [[Sistema de movimiento]] usa la categoría para determinar qué miniaturas pueden atravesar cada elemento de terreno.
- [[Terreno y visibilidad]] usa categorías, especialmente ligero y denso, para aplicar cobertura, ocultación, obstrucción y sólido.
- [[Sólido]] se aplica a elementos de terreno denso.

## Restricciones

- La categoría pertenece al elemento de terreno, no necesariamente a toda el área si esta contiene elementos distintos.
- Este artículo no sustituye a las reglas concretas de movimiento o visibilidad.

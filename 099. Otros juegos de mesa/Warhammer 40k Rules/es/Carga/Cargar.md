# Cargar

## Resumen

Cargar es el procedimiento por el que una unidad apta declara una carga, hace una tirada de carga y, si es posible, realiza un movimiento de carga.

## Definición

Una unidad apta declara una carga, tira 2D6 para determinar la distancia máxima y puede hacer un [[Movimiento de carga]] si el movimiento es posible.

## Reglas

El jugador activo resuelve cargas con sus unidades aptas de una en una.

Para declarar una carga, elige una unidad amiga que no haya declarado una carga en esta fase y que sea apta para declarar cargas.

Una unidad es apta para declarar cargas si está en el campo de batalla, salvo que otra regla indique lo contrario.

Algunas reglas impiden declarar cargas, por ejemplo:

- No hay unidades enemigas a 12" o menos.
- La unidad está trabada.
- La unidad [[Avanzar|avanzó]] o [[Retroceder|retrocedió]] este turno.

La tirada de carga se hace tirando 2D6. El resultado es la distancia máxima del [[Movimiento de carga]].

Si es posible hacer un movimiento de carga y el jugador aún quiere hacerlo, la unidad hace un [[Movimiento de carga]]. Si no, no mueve. En ambos casos, la carga queda resuelta.

## Interacciones

- [[Fase de carga]] contiene la ventana temporal normal para este procedimiento.
- [[Movimiento de carga]] contiene las condiciones de movimiento tras declarar la carga.
- [[Blanco de la carga]] identifica las unidades enemigas que deben alcanzarse.
- [[Sistema de combate]] consume el resultado de la carga, especialmente el estado [[Combatir primero]].

## Restricciones

- Una unidad no puede declarar carga si está trabada.
- Una unidad no puede declarar carga si avanzó o retrocedió este turno, salvo que otra regla lo permita.
- Este artículo no redefine el movimiento general ni el combate posterior.

# Disparo indirecto

## Resumen

Disparo indirecto permite a una unidad apta disparar armas [INDIRECTA] contra blancos que no son visibles para la miniatura atacante.

## Definición

Una unidad no trabada que no avanzó y tiene alguna arma [INDIRECTA] puede elegir el tipo de disparo indirecto.

## Reglas

La unidad debe no estar trabada, no haber avanzado este turno y tener alguna arma [INDIRECTA].

La unidad dispara como se describe en [[Atacar]].

Al disparar:

- Las armas [INDIRECTA] pueden tomar como blanco unidades que no son visibles para la miniatura atacante.
- Siempre que un arma [INDIRECTA] ataque, el blanco se beneficia de cobertura contra ese ataque.
- No puedes repetir las tiradas para impactar.
- Toda tirada para impactar de 1-5 sin modificar falla, salvo si la unidad ha [[Permanecer inmóvil|permanecido inmóvil]] y el blanco es visible para alguna unidad amiga; en ese caso, toda tirada para impactar de 1-3 sin modificar falla.

Después de disparar, hasta el final de la fase, la unidad no es apta para iniciar acciones.

## Interacciones

- [[Disparar]] aplica este tipo de disparo cuando la unidad cumple sus condiciones.
- [[Sistema de terreno y visibilidad]] mantiene las reglas de visibilidad que este tipo de disparo modifica parcialmente.
- [[Beneficiarse de cobertura]] es el estado defensivo que obtiene el blanco.
- [[Permanecer inmóvil]] forma parte de la excepción que mejora el umbral de fallo.
- [[Sistema de armas]] conserva la información de si un arma tiene [INDIRECTA].

## Restricciones

- La unidad debe no estar trabada.
- La unidad no puede haber avanzado este turno.
- Solo las armas [INDIRECTA] reciben el permiso especial de atacar blancos no visibles.
- No se pueden repetir las tiradas para impactar de estos ataques.

# Disparo de asalto

## Resumen

Disparo de asalto es el tipo de disparo que permite disparar tras avanzar, pero solo con armas [ASALTO].

## Definición

Una unidad no trabada que avanzó este turno y tiene alguna arma [ASALTO] puede disparar usando solo armas [ASALTO].

## Reglas

La unidad debe no estar trabada, haber avanzado este turno y tener alguna arma [ASALTO].

La unidad dispara como se describe en [[Atacar]].

Al disparar, solo puedes elegir armas [ASALTO] para atacar.

Después de disparar, hasta el final de la fase, la unidad no es apta para iniciar acciones.

## Interacciones

- [[Avanzar]] crea la condición que normalmente impide disparar.
- [[Disparar]] consume esta excepción cuando la unidad elige este tipo de disparo.
- [[Sistema de armas]] conserva la información de si un arma tiene [ASALTO].

## Restricciones

- La unidad debe no estar trabada.
- La unidad debe haber avanzado este turno.
- Solo se pueden elegir armas [ASALTO].
- No redefine el movimiento de avanzar ni las habilidades de arma.

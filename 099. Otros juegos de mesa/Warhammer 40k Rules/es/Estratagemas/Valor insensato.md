# Valor insensato

## Resumen

Valor insensato es una estratagema básica de 1PM que hace que una tirada de acobardamiento tenga éxito automáticamente.

## Definición

Se usa en el paso de acobardamiento de tu fase de mando, justo antes de hacer una tirada de acobardamiento para una unidad amiga.

## Reglas

Coste: 1PM.

Momento: paso de acobardamiento de tu fase de mando, justo antes de hacer una [[Tiradas de acobardamiento|tirada de acobardamiento]] para una unidad amiga.

Blanco: esa unidad.

Efecto: esa tirada de acobardamiento tiene éxito automáticamente.

## Interacciones

- [[Fase de mando]] contiene el paso de acobardamiento.
- [[Tiradas de acobardamiento]] define el resultado normal de éxito o fallo.

## Restricciones

- No puedes usar esta estratagema más de una vez por batalla.

# Repetición de mando

## Resumen

Repetición de mando es una estratagema básica de 1PM que permite repetir una tirada concreta hecha para una unidad o miniatura amiga.

## Definición

Se usa justo después de hacer una tirada permitida para una unidad o miniatura amiga.

## Reglas

Coste: 1PM.

Momento: cualquier fase, justo después de hacer una de estas tiradas para una unidad o miniatura amiga:

- avanzar;
- carga;
- daño;
- riesgo;
- impactar;
- salvación;
- herir;
- determinar la cantidad de ataques que hace un arma.

Blanco: esa unidad o miniatura.

Efecto: repite esa tirada. Si tirabas varios dados juntos, elige uno de ellos y repítelo, salvo las tiradas de carga, que se repiten en su totalidad.

## Interacciones

- [[Dados]] cubre la base de tiradas y repeticiones.
- [[Usar estratagemas]] cubre el gasto de PM y las restricciones comunes.

## Restricciones

- Solo puede aplicarse a las tiradas enumeradas.
- Una tirada de carga debe repetirse completa.

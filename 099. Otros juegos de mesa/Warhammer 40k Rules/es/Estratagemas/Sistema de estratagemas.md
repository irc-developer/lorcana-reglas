# Sistema de estratagemas

## Resumen

El Sistema de estratagemas organiza el uso de PM, las restricciones comunes de las estratagemas y las estratagemas básicas que modifican otros sistemas durante la batalla.

## Alcance

Este dominio cubre el marco general para usar estratagemas, el recurso de puntos de mando y las estratagemas básicas de las Reglas básicas.

No sustituye a los sistemas que cada estratagema invoca: disparar sigue perteneciendo a [[Sistema de disparo]], cargar a [[Sistema de carga]], combatir a [[Sistema de combate]] y las reservas a [[Sistema de reservas estratégicas]].

## Responsabilidad del dominio

- Mantener el procedimiento común de [[Usar estratagemas]].
- Mantener el glosario de [[Punto de mando (PM)]].
- Mantener las estratagemas básicas publicadas en las Reglas básicas.
- Enviar al sistema propietario cuando la duda real sea el movimiento, el disparo, la carga, el combate, la cobertura o las reservas.

# Explosivos

## Resumen

Explosivos es una estratagema básica de 1PM que permite a una unidad EXPLOSIVOS/GRANADAS infligir heridas mortales a una unidad enemiga cercana y visible.

## Definición

Se usa en tu fase de disparo con una unidad EXPLOSIVOS/GRANADAS amiga que no esté trabada, sea apta para disparar y no haya avanzado este turno.

## Reglas

Coste: 1PM.

Momento: tu fase de disparo.

Blanco: 1 unidad EXPLOSIVOS/GRANADAS amiga no trabada, apta para disparar y que no haya hecho un movimiento de avanzar este turno.

Efecto:

1. Elige 1 miniatura EXPLOSIVOS/GRANADAS de tu unidad.
2. Elige 1 unidad enemiga no trabada, a 8" o menos de esa miniatura y visible para ella.
3. Tira seis D6: por cada 4+, esa unidad enemiga sufre 1 herida mortal.

## Interacciones

- [[Sistema de disparo]] determina la ventana de uso y la aptitud para disparar.
- [[Visibilidad]] determina si la unidad enemiga puede ser elegida.
- [[Heridas mortales]] resuelve el daño producido.
- [[Avanzar]] impide usar esta estratagema con esa unidad durante el turno.

## Restricciones

- El blanco debe tener EXPLOSIVOS o GRANADAS.
- La unidad amiga no puede estar trabada.
- La unidad enemiga elegida debe estar a 8" o menos y visible para la miniatura elegida.

El uso de dispositivos electrónicos, como teléfonos, tablets o relojes inteligentes, está permitido siempre que estén en todo momento a la vista de ambos jugadores del match.

Usos permitidos incluyen:

- Usar la app **Disney Lorcana TCG Companion** para contar lore.
- Consultar la base de datos de cartas.

Un jugador puede usar su dispositivo por motivos privados, como enviar un mensaje, llamar o navegar, si primero informa a un Lore Guide para que facilite la situación.

A discreción del Lore Guide, pueden aplicarse penalizaciones como warnings, derrota de match o descalificación si un jugador no sigue estas directrices o intenta ocultar el uso de un dispositivo.

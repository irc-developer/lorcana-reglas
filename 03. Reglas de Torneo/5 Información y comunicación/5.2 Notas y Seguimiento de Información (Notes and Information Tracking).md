Los jugadores pueden usar tokens solo para seguir **su propia información pública** de juego.

No pueden usar tokens para seguir información del oponente, sea pública o privada.

Ejemplos permitidos:

- Recordarse ganar lore por una localización concreta.
- Recordarse que un personaje no puede prepararse al inicio del siguiente turno.

No se permite ninguna representación numérica en tokens para seguir información de juego salvo el **daño**.

Se recomienda llevar por escrito los totales de lore. No se permite tomar otras notas durante un match.

Consultar notas externas durante un match se considera **Outside Assistance**. Puede incluir, entre otras cosas:

- Apuntar nombres de cartas del pozo de tinta según se revelan.
- Usar dados para señalar costes de tinta de cartas reveladas.
- Usar cualquier dispositivo o proceso para crear un registro de información de juego.

Un Oficial del Torneo determina finalmente si algo cuenta como nota.

Los jugadores no pueden tomar ni consultar notas durante el Draft ni durante la construcción de mazo en Limitado.
